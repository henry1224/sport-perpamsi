# UAT Checklist v1

## Public

- [ ] Public bisa buka home tanpa login.
- [ ] Public bisa lihat jadwal per tanggal.
- [ ] Public bisa filter jadwal per cabor.
- [ ] Public bisa lihat hasil terbaru.
- [ ] Public bisa lihat bracket knockout.
- [ ] Public bisa lihat lower bracket bila format dipakai.
- [ ] Public bisa lihat klasemen medali Kontingen Provinsi.
- [ ] Public bisa lihat profil PDAM.

## Admin

- [ ] Admin bisa login.
- [ ] Admin bisa membuat event.
- [ ] Admin bisa mengelola cabor v1.
- [ ] Admin bisa mengelola PDAM, tim, atlet.
- [ ] Admin bisa import data.
- [ ] Admin bisa membuat jadwal dan match.
- [ ] Admin bisa assign panitia.
- [ ] Admin bisa finalisasi hasil.
- [ ] Admin bisa revisi hasil final dengan alasan.
- [ ] Admin bisa melihat audit log.

## Panitia

- [ ] Panitia bisa login.
- [ ] Panitia hanya melihat match tugasnya.
- [ ] Panitia bisa input hasil setelah pertandingan selesai.
- [ ] Panitia tidak bisa edit match yang bukan tugasnya.
- [ ] Panitia tidak bisa edit hasil final langsung.
- [ ] Riwayat input tercatat.

## Data dan Ranking

- [ ] Match final masuk ke bracket/klasemen.
- [ ] Peserta menang lanjut pada knockout.
- [ ] Peserta kalah masuk lower bracket bila format double elimination.
- [ ] PDAM dari provinsi sama terhubung ke Kontingen Provinsi yang sama.
- [ ] Klasemen Kontingen Provinsi mengikuti emas, perak, perunggu.
- [ ] Nama provinsi tampil sebagai peserta pada match dan bracket.
- [ ] Klasemen cabor mengikuti template ranking cabor.
