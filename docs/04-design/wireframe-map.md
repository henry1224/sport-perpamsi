# Wireframe Map v1

## Referensi Visual

- Esports Nations Cup: inspirasi event landing, dark esport mood, card besar, visual energik.
- FIFA standings: inspirasi navigasi standings/bracket dan informasi turnamen resmi.

## Public Pages

### Home

- Hero event.
- Live/Latest score cards.
- Jadwal hari ini.
- Klasemen Pimpinan Daerah ringkas.
- Highlight semifinal/final.
- Link cabor dan venue.

### Live Score / Hasil Terbaru

- Filter cabor, kategori, venue, PDAM, tanggal.
- Card match berisi status, peserta, skor, winner, waktu update.
- Karena panitia input setelah selesai, label utama: `Hasil terbaru`, bukan wajib live point-by-point.

### Jadwal

- Calendar/date tabs.
- Filter cabor, venue, PDAM.
- Status: terjadwal, selesai, final, revisi.

### Bracket

- Pilih cabor dan kategori.
- Toggle upper/lower bracket bila double elimination.
- Horizontal scroll mobile.
- Match node berisi peserta, skor, status, winner.

### Standings / Ranking

- Klasemen medali Pimpinan Daerah.
- Klasemen grup per cabor.
- Rumus ranking tampil jelas.

## Admin Pages

- Dashboard operasional.
- Master event/cabor/venue/PDAM.
- Peserta dan verifikasi.
- Jadwal dan match builder.
- Bracket manager.
- Assignment panitia.
- Audit log.
- Import/export.

## Panitia Pages

- Login.
- Daftar match tugas.
- Input hasil setelah pertandingan selesai.
- Upload/catat bukti opsional bila dibutuhkan.
- Submit hasil untuk finalisasi.
- Riwayat input hasil.

## Komponen Kunci

- Event hero.
- Match result card.
- Bracket node.
- Standing table.
- Medal ranking table.
- Score result form.
- Status badge.
- Confirmation dialog.
